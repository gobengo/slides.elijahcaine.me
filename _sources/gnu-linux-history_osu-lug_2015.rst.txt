Minix
-----

Created for educational purposes

Cheap, but not FOSS

Create by Andrew S. Tanenbaum to show the principles in Operating Systems: 
Design and Implementation (1987)

POSIX Compliant through version 2

Designed for limited resource systems


Linux
-----

.. figure:: /static/gnu-linux-history_osu-lug_2015/tux.png
    :align: right

Created by Linus Torvalds 1991

* Embedded systems
* 95% of the world's super computers
* Phones
* Servers
* Desktops

If it's got a power chord, it can run Linux.
